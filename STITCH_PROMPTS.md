## General Refinement Prompts

Use these after generating screens to dial in the design:

### Consistency Check
```
Review all generated screens and ensure they follow the DESIGN.md consistently — same navy background (#0D1B2A), same teal accent (#3A9BD5), same typography scale, same card styling, same button treatments. Fix any drift.
```

### Mobile Optimization
```
Generate mobile versions of all screens. Navigation becomes hamburger menu. Hero text scales down. Cards stack single-column. Phone number CTA stays prominent and tappable at every scroll position. Touch targets minimum 44x44px.
```

### Dark Theme Polish
```
Check all screens for dark theme issues: text contrast against navy backgrounds (minimum 4.5:1 ratio), no pure black (#000) anywhere, subtle depth through gradients not shadows, borders visible but not harsh. The site should feel atmospheric, not flat.
```

### Animation Notes (for developer handoff)
```
Add interaction notes to screens:
- Hero: content fades in and slides up on load (staggered — heading first, then subline, then CTAs, then badges)
- Cards: subtle fade-in-up on scroll into view
- Nav: slight background opacity increase on scroll
- Buttons: gentle scale + glow on hover
- Page transitions: smooth crossfade
Keep all animations under 400ms. Nothing flashy — smooth and professional.
```
